A complete Spring reservation web app for jar deployment.

## About The Project

This project demonstrate the power of java (jsp), and various data templates such as jdbctemplate, namedParameterJdbcTemplate and sessionFactory for jar deployment. 

Here's why:
* A light-weight jar file for deployment in the cloud.
* Expose the raw power of jdbc and hibernate for data.
* Using only jsp, shows javascript is not needed.
